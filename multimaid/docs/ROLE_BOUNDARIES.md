# MultiMaid Role Boundaries

## Canonical role
MultiMaid is the **band-dynamics control engine** of the Maid suite.

## It must do
- split into broad bands
- detect excess band energy
- apply per-band downward compression
- show gain reduction and band activity clearly

## It must not do
- parametric EQ graph editing
- match EQ
- full-band glue identity
- auto-correction / assistant logic
- bass enhancement identity
- reverb / delay / width
- ducking-first identity
- mastering-readiness judgment

## Neighbor plugin mapping
- FreeEQ8 = shape
- MultiMaid = control
- GlueMaid = glue
- MixMaid = assist
- BassMaid = low-end enhancement
- SpaceMaid = space
- ChainMaid = interaction
- MeterMaid = judge
